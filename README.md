# NFA from Regex Expression
A command line tool to convert a regular expression into its corresponding NFA using Thompson's construction algorithm.

### The Supported Regex Rules
1. Concatenation: AB
2. ORing: A|B or A+B
3. Repetition (0 or more): A*

### Order of Operations
1. Reperition
2. Concatenation
3. ORing
